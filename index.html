Introdução

Este material foi gerado com o estilo Corporate Editorial Design, utilizando grid assimétrico, color blocking e tipografia sans-serif. Cada seção ocupa uma página A4 na impressão.

O rodapé utiliza numeração automática via counter(page) e as cores podem ser alteradas pelas variáveis CSS para rebranding.

Coluna esquerda

Texto da primeira coluna. O layout divide o conteúdo em duas colunas com uma linha superior na cor primária, mantendo hierarquia visual e legibilidade.

Ideal para comparar tópicos ou apresentar conteúdo em paralelo.

Coluna direita

Texto da segunda coluna. O design segue as diretrizes de Web-to-Print com tamanho A4 (210mm × 297mm) e quebra de página automática.

Use npm run pdf para gerar o PDF a partir deste HTML.
